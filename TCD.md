# Tableaux Croisés Dynamiques

## Définition
Un TCD pour tableaux croisés dynamiques permet de représenter les informations d'un tableau Excel, sous un autre angle.

## Règles préalables
Il existe des règles préalables à appliquer sur un tableau de données, avant de croiser les données.

### Règle 1
Le tableau doit être entièrement rempli. Aucune ligne, colonne ou cellule vide.  
Pas de cellules fusionnées.

### Règle 2
Pas de ligne de sous-total.

### Règle 3
Les données doivent être organisées en colonne. Mettre un en-tête pour chaque colonne.

### Règle 4
Une seule ligne d'en-tête. Le nom d'en-tête est unique. 
Exemple : ne pas avoir deux colonnes portant le nom __Ventes__.

### Règle 5
Transformer les données à traiter sous forme de tableau (CTRL+L ou CTRL+T).
Retirer éventuellement les mises en forme.

__Procédure :__
- Sélectionner une cellule quelconque du tableau
- Aller à Menu -> Accueil -> Mettre sous forme de tableau
- Choisir une mise en page
- Cocher la case "Mon tableau comporte des en-têtes"
